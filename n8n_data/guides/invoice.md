# 청구서/세금계산서 문의 응대 가이드

세금계산서는 사업자만 발급 가능하며, 발급 기한은 당월분 익월 10일까지입니다.

## 세금계산서 발급

**문의**: 세금계산서를 발급해주세요

**답변 방법**:
- 사업자 정보 수집 (사업자등록번호, 상호, 대표자명, 사업장 주소, 업태/종목)
- 발급 가능 여부 확인 (결제 완료 건인지)
- 발급 일정 안내 (영업일 기준 2-3일)

**답변 예시**:
"세금계산서 발급을 위해 다음 정보를 회신해 주시기 바랍니다: 사업자등록번호, 상호명, 대표자명, 사업장 주소, 업태/종목, 이메일 주소. 정보 확인 후 영업일 기준 2-3일 내 국세청 전자세금계산서로 발급해 드리겠습니다."

## 영수증 발급

**문의**: 영수증이 필요해요

**답변 방법**:
- 자동 발급 시스템 안내 (고객 포털 > 결제 내역 > 영수증 다운로드)
- 수동 발급 시 이메일 주소 확인
- 즉시 다운로드 vs 이메일 발송 선택

**답변 예시**:
"영수증은 고객 포털 > 결제 내역 메뉴에서 즉시 다운로드 가능합니다. 이메일로 받으시려면 수신 희망 이메일 주소를 회신해 주시면 1-2영업일 내 발송해 드리겠습니다."

## 세금계산서 수정/재발급

**문의**: 발급된 세금계산서를 수정하고 싶어요

**답변 방법**:
- 수정 사유 확인 (정보 오류 vs 금액 변경)
- 단순 정보 수정은 원본 취소 후 재발급 (2-3영업일)
- 금액 변경은 수정세금계산서 발급 사유서 필요 (3-5영업일)

**답변 예시**:
"세금계산서 수정을 위해 원본을 국세청에서 취소 처리 후 재발급합니다. 단순 정보 수정은 2-3영업일, 금액 변경 시에는 수정 사유서가 필요하며 3-5영업일이 소요됩니다. 수정이 필요한 항목을 알려주시기 바랍니다."

## 거래명세서 발급

**문의**: 거래명세서가 필요해요

**답변 방법**:
- 건별 명세서 vs 월 합산 명세서 선택
- 상세 품목 내역 포함 여부 확인
- Excel 또는 PDF 형식 선택

**답변 예시**:
"거래명세서는 건별 또는 월 합산으로 발급 가능합니다. 발급 희망 기간과 수신 이메일 주소를 회신해 주시면 Excel 또는 PDF 형식으로 발송해 드리겠습니다."

## 발급 내역 조회/재전송

**문의**: 과거 세금계산서를 다시 받고 싶어요

**답변 방법**:
- 고객 포털에서 직접 조회 가능 안내 (결제 관리 > 세금계산서 내역)
- 재전송 시 발급일, 금액, 수신 이메일 확인
- 1일 내 재전송 처리

**답변 예시**:
"과거 발급 내역은 고객 포털 > 결제 관리 > 세금계산서 내역에서 조회 및 다운로드 가능합니다. 재전송이 필요하시면 발급일과 금액을 알려주시면 1영업일 내 재전송해 드리겠습니다."

## 해외 사업자 Invoice

**문의**: 영문 Invoice가 필요해요

**답변 방법**:
- Company Name, VAT ID, Registered Address 수집
- 통화 선택 (USD/EUR/KRW)
- 영문 Invoice 발급 (2-3영업일)

**답변 예시**:
"영문 Invoice 발급을 위해 Company Name, VAT ID, Registered Address, Contact Email을 회신해 주시기 바랍니다. 통화는 USD, EUR, KRW 중 선택 가능하며, 2-3영업일 내 이메일로 발송해 드리겠습니다."

## 발급 불가 케이스

**문의**: 세금계산서 발급이 안 된다고 하는데요?

**답변 방법**:
- 미결제 건, 무료 플랜, 환불 완료 건은 발급 불가
- 발급 불가 사유 명확히 설명
- 가능한 대안 제시 (거래 확인서 등)

**답변 예시**:
"해당 거래는 [미결제/무료 플랜/환불 완료]로 세금계산서 발급이 불가능합니다. 거래 확인서가 필요하시면 별도 발급 가능하니 알려주시기 바랍니다."
