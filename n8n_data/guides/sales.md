# 영업/세일즈 문의 응대 가이드

엔터프라이즈 고객은 맞춤 견적 제공, 14일 무료 체험 권장, ROI 계산기 활용을 원칙으로 합니다.

## 견적 요청

**문의**: 견적서를 받고 싶어요

**답변 방법**:
- 사용 규모 파악 (사용자 수, 데이터량, 필요 기능)
- 계약 기간 및 예산 범위 확인
- 맞춤 견적서 발송 (1-2영업일)

**답변 예시**:
"정확한 견적을 위해 예상 사용자 수, 필요 기능, 희망 계약 기간(월간/연간)을 회신해 주시기 바랍니다. 정보 확인 후 1-2영업일 내 맞춤 견적서를 발송해 드리겠습니다. 표준 가격은 Basic $29/월, Pro $99/월, Enterprise는 맞춤 견적입니다."

## 데모/시연 요청

**문의**: 제품 데모를 보고 싶어요

**답변 방법**:
- 라이브 데모(30-45분) vs 사전 녹화 데모 선택
- 희망 일정 조율 (3개 후보 일정 수집)
- 참석 인원, 중점 확인 기능 파악

**답변 예시**:
"라이브 데모는 30-45분 소요되며 실시간 Q&A가 가능합니다. 희망 날짜/시간대를 3개 제시해 주시면 일정을 조율하겠습니다. 사전 녹화 데모 영상은 [URL]에서 즉시 확인 가능합니다."

## 무료 체험 연장 요청

**문의**: 무료 체험을 더 사용하고 싶어요

**답변 방법**:
- 기본 14일, 1회 한정 7일 연장 가능
- 의사결정 진행 상황 및 추가 확인 필요 기능 파악
- 대안으로 30일 환불 정책 안내

**답변 예시**:
"무료 체험은 기본 14일이며, 의사결정 진행 중이시면 7일 추가 연장 가능합니다. 현재 평가 진행 상황과 추가로 확인이 필요한 기능을 알려주시면 연장 처리해 드리겠습니다."

## 플랜 비교/컨설팅

**문의**: 어떤 플랜이 적합한가요?

**답변 방법**:
- 현재 사용자 수, 데이터량, 주요 사용 기능 파악
- 플랜별 기능 비교표 제공
- ROI 계산 및 최적 플랜 추천

**답변 예시**:
"현재 사용자 7명, 데이터 50GB 예상이시면 Pro 플랜($99/월)을 추천합니다. Pro는 10명 사용자, 100GB 저장공간, 우선 지원을 제공하며 연간 결제 시 $200 절감됩니다. 상세 비교표는 [URL]에서 확인 가능합니다."

## 계약/구매 절차

**문의**: 구매는 어떻게 하나요?

**답변 방법**:
- Self-Service(즉시 구매) vs Enterprise(맞춤 계약) 구분
- 필요 서류 및 결제 방법 안내
- 처리 기간 안내 (즉시 vs 2-4주)

**답변 예시**:
"일반 플랜은 웹사이트에서 플랜 선택 후 신용카드로 즉시 구매 가능합니다. 엔터프라이즈 계약은 견적서 발행 → 계약 체결 → 온보딩 순으로 진행되며 2-4주 소요됩니다."

## 경쟁사 비교

**문의**: 다른 제품과 비교하고 싶어요

**답변 방법**:
- 주요 경쟁사와의 객관적 비교표 제공
- 차별화 포인트 설명 (가격, 기능, 지원)
- 마이그레이션 지원 안내

**답변 예시**:
"당사는 동급 대비 20-30% 저렴하며 24/7 실시간 지원을 제공합니다. G2 평점 4.7/5.0으로 고객 만족도가 높습니다. 상세 비교표 [PDF]와 무료 체험으로 직접 비교해 보시기 바랍니다. 경쟁사에서 전환 시 무료 마이그레이션과 첫 달 50% 할인을 제공합니다."

## 대량 라이선스/볼륨 할인

**문의**: 대량 구매 시 할인이 있나요?

**답변 방법**:
- 수량별 할인율 제시 (11-50명 15%, 51-100명 25%)
- 계약 기간별 추가 할인 (연간 17%, 2-3년 추가 5-10%)
- 예시 견적 제공

**답변 예시**:
"100 라이선스 구매 시 25% 볼륨 할인이 적용되어 월 $74/라이선스입니다. 2년 계약 시 추가 5% 할인으로 최종 월 $70/라이선스, 연간 $84,000입니다. 정확한 견적을 위해 sales@company.com으로 연락 주시기 바랍니다."

## 재계약/갱신

**문의**: 계약 갱신하고 싶어요

**답변 방법**:
- 현재 계약 정보 및 사용 현황 제공
- 갱신 옵션 제시 (현재 플랜 유지, 연간 전환, 업그레이드)
- 얼리버드 갱신 혜택 안내 (만료 30일 전 갱신 시 추가 5% 할인)

**답변 예시**:
"현재 Pro 플랜 월간 계약이 [날짜]에 만료됩니다. 연간 계약으로 전환 시 17% 할인($990/년)되며, 만료 30일 전 갱신 시 추가 5% 할인이 적용됩니다. 갱신 옵션을 선택해 주시면 처리해 드리겠습니다."
