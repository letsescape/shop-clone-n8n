# 결제 문의 응대 가이드

결제 정보는 보안상 이메일로 직접 공유하지 않으며, 24시간 이내 처리를 원칙으로 합니다.

## 결제 실패

**문의**: 결제가 안 돼요 / 카드가 거부당했어요

**답변 방법**:
- 카드사 확인 요청 (한도, 유효기간, 해외결제 차단 여부)
- 다른 결제 수단 제안
- 입력 정보 정확성 확인 (카드번호, CVV, 주소)

**답변 예시**:
"결제 실패는 카드 한도 초과, 유효기간 만료, 해외결제 차단 등이 원인일 수 있습니다. 카드사에 확인하시고, 다른 결제 수단(PayPal 등)을 시도해 주시기 바랍니다. 문제가 지속되면 입력 정보를 다시 확인해 주시기 바랍니다."

## 이중 결제

**문의**: 같은 금액이 2번 청구됐어요

**답변 방법**:
- 거래 내역 확인 (임시 승인 vs 실제 결제 구분)
- 임시 승인은 3-5영업일 내 자동 취소 안내
- 실제 중복 결제 시 즉시 환불 처리

**답변 예시**:
"거래 내역을 확인한 결과 1건은 임시 승인으로 3-5영업일 내 자동 취소됩니다. 실제 중복 결제가 확인되면 즉시 환불 처리하며 3-5영업일 내 계좌로 입금됩니다."

## 환불 요청

**문의**: 환불해주세요

**답변 방법**:
- 환불 정책 확인 (30일 이내 전액 환불)
- 환불 사유 확인 (서비스 개선 피드백 수집)
- 환불 절차 및 소요 기간 안내

**답변 예시**:
"30일 이내 구매 건은 전액 환불 가능합니다. 환불 신청서를 작성해 주시면 1-2영업일 검토 후 환불 처리되며, 3-5영업일 내 계좌로 입금됩니다. 환불 사유를 알려주시면 서비스 개선에 반영하겠습니다."

## 영수증/세금계산서 발급

**문의**: 영수증이나 세금계산서가 필요해요

**답변 방법**:
- 영수증: 고객 포털에서 즉시 다운로드 안내
- 세금계산서: 사업자 정보 수집 후 발급 (2-3영업일)
- 자동 발급 시스템 vs 수동 요청 구분

**답변 예시**:
"영수증은 고객 포털 > 결제 내역에서 즉시 다운로드 가능합니다. 세금계산서는 사업자등록번호, 상호명, 대표자명을 회신해 주시면 2-3영업일 내 발급해 드리겠습니다."

## 결제 수단 변경

**문의**: 등록된 카드를 변경하고 싶어요

**답변 방법**:
- 고객 포털에서 직접 변경 가능 안내
- 결제 정보 보안 강조
- 변경 후 다음 결제 주기부터 적용

**답변 예시**:
"결제 수단은 고객 포털 > 결제 관리 > 결제 수단 변경에서 직접 수정하실 수 있습니다. 보안을 위해 전체 카드 번호는 표시되지 않으며, 변경 후 다음 결제 주기부터 적용됩니다."

## 자동 결제 관리

**문의**: 자동 결제를 중지하고 싶어요

**답변 방법**:
- 구독 취소 vs 자동 갱신 중지 구분
- 구독 취소 시 현재 기간 종료일까지 이용 가능
- 데이터 보관 기간 안내 (30일)

**답변 예시**:
"자동 결제 중지는 고객 포털 > 구독 관리에서 '자동 갱신 중지'를 클릭하시면 됩니다. 현재 결제 주기 종료일까지 서비스 이용 가능하며, 이후 자동으로 갱신되지 않습니다. 데이터는 30일간 보관됩니다."

## 결제 내역 조회

**문의**: 과거 결제 내역을 확인하고 싶어요

**답변 방법**:
- 고객 포털에서 전체 내역 조회 가능 안내
- 기간별 필터링 및 Excel 다운로드 기능 안내
- 특정 거래 내역 재전송 가능

**답변 예시**:
"결제 내역은 고객 포털 > 결제 관리에서 전체 조회 가능하며, 기간별 필터링과 Excel 다운로드가 지원됩니다. 특정 거래 영수증이 필요하시면 거래일과 금액을 알려주시면 재전송해 드리겠습니다."
