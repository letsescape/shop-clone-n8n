# 기타 문의 응대 가이드

명확한 분류가 어려운 문의나 일반적인 문의를 처리합니다.

## 분류 불명확 문의

**문의**: [명확하지 않은 문의]

**답변 방법**:
- 추가 정보 요청으로 문의 내용 구체화
- 주제 선택지 제공 (사용법, 기술 문제, 결제, 계정, 기능 요청)
- 적절한 카테고리로 재분류 후 담당 부서 연결

**답변 예시**:
"더 정확한 지원을 위해 문의 주제를 선택해 주시기 바랍니다: 1) 제품 사용법, 2) 기술적 문제, 3) 결제/청구, 4) 계정 관련, 5) 기능 요청. 추가로 문제 발생 시점과 상세 내용을 회신해 주시면 담당 부서에서 신속히 처리하겠습니다."

## 일반 문의

**문의**: 서비스에 대해 알고 싶어요

**답변 방법**:
- 도움말 센터, FAQ, 비디오 튜토리얼 링크 제공
- 커뮤니티 포럼, Discord 서버 안내
- 자주 묻는 질문 요약 제공

**답변 예시**:
"도움말 센터 [URL]에서 사용 가이드와 FAQ를 확인하실 수 있습니다. 무료 체험은 14일 가능하며, 결제 방법은 신용카드, PayPal이 지원됩니다. 추가 문의사항이 있으시면 언제든 연락 주시기 바랍니다."

## 피드백/제안

**문의**: 이런 기능이 있으면 좋겠어요

**답변 방법**:
- 피드백에 감사 표시
- 제품 팀 전달 약속
- 유사 기능이나 대안 제시
- 피드백 포럼/투표 시스템 안내

**답변 예시**:
"소중한 제안 감사합니다. 제품 팀에 전달하여 향후 개발 우선순위에 반영하겠습니다. 현재는 [대안 방법]으로 유사한 기능을 사용하실 수 있습니다. 피드백 포럼 [URL]에서 투표하시면 개발에 더 빠르게 반영됩니다."

## 불만/컴플레인

**문의**: 서비스가 불만족스러워요

**답변 방법**:
- 공감과 사과 표시
- 문제 상황 정확히 파악
- 즉각 조치 및 보상 제시
- 담당자 배정 및 후속 조치 약속

**답변 예시**:
"불편을 끼쳐드려 대단히 죄송합니다. 즉시 [조치 내용]을 진행했으며, 불편에 대한 보상으로 [1개월 무료 연장/크레딧 제공]을 제공해 드리겠습니다. 담당자가 [처리 기한]까지 해결하여 연락드리겠습니다."

## 칭찬/감사

**문의**: 서비스가 정말 좋아요!

**답변 방법**:
- 감사 인사
- 팀 공유 약속
- 리뷰 작성 요청 (선택)

**답변 예시**:
"따뜻한 말씀 감사합니다! 전 팀원과 공유하여 더 나은 서비스를 제공하는 동기로 삼겠습니다. 시간이 되신다면 G2 [URL]에 리뷰를 남겨주시면 다른 고객들에게도 큰 도움이 됩니다."

## 채용/인재 문의

**문의**: 채용 공고가 있나요?

**답변 방법**:
- 채용 페이지 링크 제공
- 현재 채용 중인 포지션 소개
- 지원 방법 및 절차 안내

**답변 예시**:
"채용 페이지 [URL]에서 현재 채용 중인 포지션을 확인하실 수 있습니다. 관심 있는 포지션에 '지원하기'를 클릭하여 이력서를 제출해 주시기 바랍니다. 자세한 문의는 careers@company.com으로 연락 주시기 바랍니다."

## 법적 문의

**문의**: 이용약관/개인정보 관련 문의가 있어요

**답변 방법**:
- 법무팀 즉시 연결
- 공식 법적 문서 링크 제공 (이용약관, 개인정보 처리방침)
- 응답 기한 안내 (일반 5-7영업일, 긴급 2영업일)

**답변 예시**:
"법적 문의는 legal@company.com으로 연락 주시기 바랍니다. 이용약관 [URL]과 개인정보 처리방침 [URL]을 참고하실 수 있으며, 법무팀이 5-7영업일 내 회신드리겠습니다."

## 보안 취약점 신고

**문의**: 보안 문제를 발견했어요

**답변 방법**:
- 즉시 보안팀 에스컬레이션
- 보안 신고 전용 채널 안내 (security@company.com)
- 책임 있는 공개 정책 안내
- 버그 바운티 프로그램 안내

**답변 예시**:
"보안 취약점 신고 감사합니다. 보안팀에 즉시 전달되며 24시간 내 확인 연락드리겠습니다. 상세 내용은 security@company.com으로 회신해 주시기 바랍니다. 심각도에 따라 최대 $5,000의 버그 바운티를 지급합니다."
