# 계정 문의 응대 가이드

계정 관련 문의는 본인 인증 필수이며, 개인정보 보호를 최우선으로 합니다.

## 비밀번호 재설정

**문의**: 비밀번호를 잊어버렸어요 / 재설정하고 싶어요

**답변 방법**:
- 자동 재설정 링크 발송 안내 (로그인 페이지 → "비밀번호 찾기")
- 이메일 접근 불가 시 본인 인증 절차 안내 (계정 생성일, 최근 결제 내역 확인)
- 재설정 후 2단계 인증 활성화 권장

**답변 예시**:
"비밀번호 재설정은 로그인 페이지에서 '비밀번호 찾기'를 클릭하시면 등록된 이메일로 재설정 링크가 발송됩니다. 이메일에 접근할 수 없으시면 본인 확인을 위해 계정 생성일과 최근 결제 내역을 회신해 주시기 바랍니다."

## 계정 잠금/정지

**문의**: 계정이 잠겼어요 / 로그인이 안 돼요

**답변 방법**:
- 잠금 사유 명확히 설명 (의심스러운 로그인, 약관 위반, 결제 실패)
- 해제 조건과 절차 안내
- 임시 잠금(24시간 자동 해제) vs 수동 해제 필요 구분

**답변 예시**:
"계정이 [의심스러운 로그인 시도/결제 실패]로 인해 임시 잠금 처리되었습니다. 본인 인증을 위해 이메일로 회신해 주시면 검토 후 1-2영업일 내 해제해 드리겠습니다."

## 이메일 변경

**문의**: 계정 이메일을 변경하고 싶어요

**답변 방법**:
- 현재 이메일과 새 이메일 모두 인증 필요
- 변경 후 모든 기기에서 자동 로그아웃 안내
- 연동된 서비스 로그인 정보도 업데이트 필요

**답변 예시**:
"이메일 변경 시 현재 이메일과 새 이메일 모두 인증이 필요합니다. 변경 후 모든 기기에서 자동 로그아웃되며, 연동된 서비스의 로그인 정보도 업데이트하셔야 합니다. 진행을 원하시면 새 이메일 주소를 회신해 주시기 바랍니다."

## 계정 삭제

**문의**: 계정을 삭제하고 싶어요

**답변 방법**:
- 삭제 시 모든 데이터 영구 소실 경고
- 30일 유예 기간 안내 (복구 가능)
- 대안으로 비활성화 옵션 제시

**답변 예시**:
"계정 삭제 시 모든 데이터가 영구적으로 삭제되며, 30일 유예 기간 이후에는 복구가 불가능합니다. 계정을 비활성화하시면 데이터를 보관하면서 언제든 재활성화할 수 있습니다. 삭제를 확정하시려면 '계정 삭제를 요청합니다'라고 회신해 주시기 바랍니다."

## 2단계 인증 (2FA) 문제

**문의**: 2FA 기기를 분실했어요 / 인증 코드가 안 와요

**답변 방법**:
- 백업 코드 사용 안내
- 백업 코드 분실 시 본인 인증 후 재설정
- 새 기기 등록 절차 안내

**답변 예시**:
"2FA 기기 분실 시 계정 생성 시 제공된 8자리 백업 코드를 사용하실 수 있습니다. 백업 코드도 분실하셨다면 본인 인증을 위해 계정 생성일과 등록된 전화번호 끝 4자리를 회신해 주시기 바랍니다."

## 프로필 정보 변경

**문의**: 이름/전화번호 등을 변경하고 싶어요

**답변 방법**:
- 일반 정보는 고객 포털에서 직접 변경 가능
- 중요 정보(실명, 사업자번호)는 서류 제출 필요
- 변경 승인 시간 안내 (즉시 vs 1-2영업일)

**답변 예시**:
"표시 이름, 프로필 사진 등은 고객 포털 > 내 정보에서 직접 변경 가능합니다. 실명 변경이나 사업자 정보 변경은 신분증 또는 사업자등록증 사본이 필요하며, 검토 후 1-2영업일 내 처리됩니다."

## 계정 병합

**문의**: 여러 계정을 하나로 합치고 싶어요

**답변 방법**:
- 기술팀 검토 필요 (5-7영업일 소요)
- 완전 병합 vs 수동 이전 vs 계정 연결 옵션 설명
- 데이터 이전 가능 범위 안내

**답변 예시**:
"계정 병합은 기술팀 검토가 필요하며 5-7영업일이 소요됩니다. 고객님이 직접 데이터를 내보내기/가져오기 하시는 수동 이전 방법도 가능합니다. 선호하시는 방법을 알려주시면 자세히 안내드리겠습니다."
