# 마케팅 문의 응대 가이드

프로모션, 이벤트, 제휴, 광고 등 마케팅 관련 문의를 처리합니다.

## 프로모션 코드/할인 쿠폰

**문의**: 프로모션 코드를 어떻게 사용하나요?

**답변 방법**:
- 코드 적용 방법 단계별 안내 (플랜 선택 → 코드 입력 → 할인 확인)
- 코드 정보 제공 (할인율, 유효기간, 적용 조건)
- 적용 안 되는 경우 원인 설명 (만료, 조건 미충족, 중복 사용)

**답변 예시**:
"프로모션 코드는 플랜 선택 페이지에서 '프로모션 코드 입력'을 클릭하여 적용하실 수 있습니다. 해당 코드는 [유효기간], [최소 구매 금액] 조건이 있으며, 다른 프로모션과 중복 적용되지 않습니다."

## 이벤트/캠페인 참여

**문의**: 진행 중인 이벤트가 있나요?

**답변 방법**:
- 현재 진행 중인 이벤트 소개 (기간, 대상, 혜택)
- 참여 방법 상세 안내
- 당첨자 발표 일정 및 방법 안내

**답변 예시**:
"현재 [이벤트명]이 진행 중입니다 (기간: [날짜]). 참여 방법은 [가입/구매/공유]하시고 이벤트 페이지에서 응모하시면 됩니다. 당첨자는 [발표일]에 이메일로 개별 통보됩니다."

## 제휴/파트너십 문의

**문의**: 제휴를 제안하고 싶어요

**답변 방법**:
- 제휴 형태 확인 (리셀러, 기술 파트너, 콘텐츠 파트너)
- 각 파트너십 조건과 혜택 설명
- 신청 절차 및 필요 정보 안내

**답변 예시**:
"제휴 프로그램은 리셀러, 기술 파트너, 콘텐츠 파트너로 운영됩니다. 파트너십 신청서 작성 후 담당자가 검토하며 7-10영업일이 소요됩니다. 회사명, 사업자등록번호, 웹사이트, 예상 고객 수를 회신해 주시기 바랍니다."

## 광고/스폰서십

**문의**: 광고를 집행하고 싶어요

**답변 방법**:
- 광고 상품 소개 (디스플레이, 콘텐츠 스폰서십, 이벤트 스폰서)
- 광고 목적, 타겟, 예산 확인
- 마케팅팀 연결 안내

**답변 예시**:
"광고 상품으로 디스플레이 광고, 콘텐츠 스폰서십, 이벤트 스폰서십이 있습니다. 광고 목적, 타겟 고객, 예산 범위를 회신해 주시면 담당자가 1-2영업일 내 연락드리겠습니다. 미디어킷은 [URL]에서 다운로드 가능합니다."

## 뉴스레터 구독 관리

**문의**: 뉴스레터를 받고 싶어요 / 구독 취소하고 싶어요

**답변 방법**:
- 구독 신청 방법 안내 (구독 페이지, 이메일 확인)
- 구독 취소 방법 안내 (뉴스레터 하단 링크)
- 수신 설정 변경 가능 안내 (주간/월간 선택)

**답변 예시**:
"뉴스레터 구독은 [URL]에서 이메일 주소를 입력하시면 확인 이메일이 발송됩니다. 구독 취소는 뉴스레터 하단의 '구독 취소' 링크를 클릭하시면 즉시 처리됩니다."

## 소셜 미디어/커뮤니티

**문의**: 공식 소셜 미디어 채널이 있나요?

**답변 방법**:
- 공식 채널 링크 제공 (Facebook, Twitter, LinkedIn 등)
- 커뮤니티 포럼, Discord 서버 안내
- 커뮤니티 가이드라인 및 참여 방법 안내

**답변 예시**:
"공식 채널은 Facebook [URL], Twitter [URL], 사용자 포럼 [URL]이 있습니다. 커뮤니티 가이드라인을 확인하신 후 자유롭게 질문과 경험을 공유해 주시기 바랍니다."

## 리뷰/추천 요청

**문의**: 리뷰를 작성하면 혜택이 있나요?

**답변 방법**:
- 리뷰 플랫폼 안내 (G2, Capterra, 자사 웹사이트)
- 리뷰 인센티브 제공 안내 (기프트 카드, 무료 업그레이드)
- 케이스 스터디 참여 혜택 안내

**답변 예시**:
"리뷰 작성 시 $20 Amazon 기프트 카드 또는 1개월 무료 업그레이드를 제공합니다. 리뷰 플랫폼에 작성 후 스크린샷을 첨부하여 회신해 주시면 3-5영업일 내 지급해 드리겠습니다."

## 언론/미디어 문의

**문의**: 취재/인터뷰를 요청하고 싶어요

**답변 방법**:
- PR 팀 연결 안내
- 프레스킷 다운로드 링크 제공
- 필요 정보 수집 (매체명, 기사 주제, 마감일)

**답변 예시**:
"언론 문의는 press@company.com으로 연락 주시기 바랍니다. 프레스킷은 [URL]에서 다운로드 가능합니다. 매체명, 기사 주제, 마감일을 회신해 주시면 PR 팀이 1-2영업일 내 연락드리겠습니다."
