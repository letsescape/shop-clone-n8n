# 구독 문의 응대 가이드

구독 변경은 결제 주기에 따라 즉시 또는 차기 결제일부터 적용되며, 다운그레이드 시 데이터 손실 가능성을 안내합니다.

## 구독 업그레이드

**문의**: 플랜을 업그레이드하고 싶어요

**답변 방법**:
- 즉시 업그레이드 vs 차기 결제일 업그레이드 선택 제공
- 차액 정산 방식 설명 (일할 계산)
- 업그레이드 후 새 기능 및 제한 해제 안내

**답변 예시**:
"즉시 업그레이드 시 차액만큼 결제 후 바로 반영되며, 차기 결제일 업그레이드는 [날짜]부터 자동 변경됩니다. 업그레이드 시 [무제한 사용자, 고급 기능, 우선 지원] 등의 혜택을 즉시 이용하실 수 있습니다."

## 구독 다운그레이드

**문의**: 플랜을 다운그레이드하고 싶어요

**답변 방법**:
- 데이터 손실 위험 경고 (저장 용량 초과 시 읽기 전용 전환)
- 기능 제한 명확히 안내
- 다운그레이드 전 백업 권장
- 차기 결제일부터 자동 변경

**답변 예시**:
"다운그레이드 시 저장 용량이 [100GB → 10GB]로 축소되며, 초과 데이터는 읽기 전용으로 전환됩니다. 차기 결제일([날짜])부터 자동 변경되며, 현재 기간은 기존 플랜을 유지합니다. 중요 데이터는 사전 백업을 권장합니다."

## 구독 취소

**문의**: 구독을 취소하고 싶어요

**답변 방법**:
- 취소 사유 확인 (서비스 개선 피드백 수집)
- 현재 결제 주기 종료일까지 이용 가능 안내
- 데이터 보관 기간 안내 (30일)
- 재구독 시 복구 가능 안내

**답변 예시**:
"구독 취소 시 현재 결제 주기 종료일([날짜])까지 서비스 이용 가능하며, 자동 갱신이 중단됩니다. 데이터는 30일간 보관되어 재구독 시 복구 가능합니다. 불편하셨던 점을 알려주시면 서비스 개선에 반영하겠습니다."

## 결제 실패로 인한 구독 중단

**문의**: 결제 실패로 서비스가 중단됐어요

**답변 방법**:
- 실패 사유 설명 (카드 만료, 잔액 부족)
- 유예 기간 안내 (3-7일)
- 결제 수단 업데이트 방법 안내
- 재활성화 절차 안내

**답변 예시**:
"결제 실패 사유는 [카드 만료/잔액 부족]입니다. [날짜]까지 유예 기간이 있으며, 고객 포털에서 결제 수단을 업데이트하시면 즉시 재활성화됩니다. 유예 기간 경과 시 구독이 자동 취소됩니다."

## 플랜 비교 문의

**문의**: 어떤 플랜이 맞는지 모르겠어요

**답변 방법**:
- 사용 패턴 파악 (사용자 수, 데이터량, 필요 기능)
- 플랜별 차이점 명확히 설명
- 비용 효율성 비교 및 추천

**답변 예시**:
"현재 사용자 7명, 데이터 50GB 예상이시면 Pro 플랜을 추천합니다. Pro는 10명까지, 100GB, 우선 지원을 제공하며 연간 결제 시 17% 할인됩니다. Basic은 소규모 팀용, Enterprise는 대규모 조직용입니다. 상세 비교표: [URL]"

## 구독 일시 정지

**문의**: 구독을 잠깐 멈췄다가 다시 시작하고 싶어요

**답변 방법**:
- 일시 정지 기능 지원 여부 확인 (일반적으로 미지원)
- 대안 제안 (구독 취소 후 재가입, 다운그레이드)
- 데이터 보관 기간 안내

**답변 예시**:
"구독 일시 정지 기능은 지원되지 않습니다. 대안으로 구독을 취소하시면 30일간 데이터가 보관되어 재가입 시 복구 가능합니다. 또는 최소 플랜으로 다운그레이드하여 저렴하게 유지하실 수 있습니다."

## 구독 주기 변경

**문의**: 월간에서 연간으로 변경하고 싶어요

**답변 방법**:
- 월간 → 연간 변경 시 할인율 안내 (17%)
- 변경 시점 선택 (즉시 vs 차기 결제일)
- 환불 및 차액 정산 방식 설명

**답변 예시**:
"월간 $99에서 연간 $990으로 변경 시 17% 할인됩니다. 즉시 변경 시 남은 월간 구독 금액을 일할 계산하여 연간 결제에서 차감합니다. 차기 결제일([날짜])부터 변경도 가능합니다."
